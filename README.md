# Weather Dashboard

A real-time Weather Dashboard built with HTML, CSS, and JavaScript that fetches live weather data from a RESTful API using asynchronous JavaScript.

## Features

* Fetch weather data using Fetch API
* Async/Await implementation
* REST API integration
* JSON parsing and rendering
* Search weather by city name
* Temperature display
* Humidity display
* Wind speed display
* Weather condition display
* Comprehensive error handling
* Responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* Async/Await
* OpenWeatherMap API

## Setup Instructions

1. Clone the repository.

```bash
git clone https://github.com/your-username/weather-dashboard.git
```

2. Create a free account at OpenWeatherMap.

3. Generate an API key.

4. Replace:

```javascript
const API_KEY = "YOUR_API_KEY";
```

with your actual API key.

5. Open `index.html` in your browser.
